# Quick Start Guides

This directory contains quick start guides and getting started documentation for the FastAPI Service boilerplate.

## Purpose

Quick start guides help developers:
- Get started quickly
- Run the service locally
- Make their first API call
- Understand basic concepts

## Contents

- **Getting Started**: Basic setup and running instructions
- **Quick Start**: Fastest way to get the service running
- **First Steps**: First steps after setup

## Usage

Start here if you're:
- New to the project
- Setting up for the first time
- Looking for quick setup instructions
