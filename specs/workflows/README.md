# Workflows

This directory contains workflow specifications and templates for the FastAPI Service boilerplate.

## Purpose

Workflow specifications define:
- Git workflow and branching strategies
- Pull request process
- Issue tracking templates
- Code review process

## Contents

- **GitHub Workflow**: Git workflow conventions and branching strategies
- **PR Template**: Pull request template for code reviews
- **Issue Templates**: Templates for creating GitHub issues

## Usage

These workflows help maintain:
- Consistent development process
- Proper code review practices
- Clear issue tracking
- Version control best practices
