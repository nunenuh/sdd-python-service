# API Examples

This directory contains example requests and responses for the FastAPI Service API.

## Purpose

Examples demonstrate how to interact with the API by providing:
- Real-world request examples
- Expected response formats
- Error handling scenarios
- Authentication examples

## Subdirectories

- **`requests/`** - Example HTTP requests showing how to call API endpoints, including headers, query parameters, and request bodies
- **`responses/`** - Example API responses showing the structure and format of successful and error responses

## Usage

These examples are used for:
- **Documentation** - Help developers understand how to use the API
- **Testing** - Serve as test cases and validation examples
- **Client Generation** - Guide API client code generation
- **Onboarding** - Help new developers understand the API quickly
